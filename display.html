Display

The display property specifies the type of box used for an element.

Possible Values

  • none
  • inline
  • block
  • inline-block

p.ex1 {display: none;}

p.ex2 {display: inline;}

p.ex3 {display: block;}

p.ex4 {display: inline-block;}
